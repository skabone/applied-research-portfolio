## Limitations

- Survey results are descriptive and represent only respondents.
- Registration export fields vary across events; tables are built from available columns only.
- Open-ended feedback was intentionally excluded from the public dataset in this first pass.

## Recommendations (First Pass)

Based on the patterns above (and keeping sample sizes in mind):

1. **Increase student turnout / qualified candidate volume:** add targeted outreach via high-activation channels (email + faculty announcements) and add a “prep checklist” link in promotions.
2. **Improve employer experience ergonomics:** address long-duration standing and table layout feedback with explicit accessibility options (chairs/space).
3. **Strengthen student preparedness:** add a short pre-event “how to talk to employers” micro-module and promote CCC services earlier in the cycle.
