# Career Fair Program Analytics — Project Summary

**Author:** Mintay Misgano, PhD  
**Context:** Program evaluation + stakeholder analytics for a university career fair

## What this project is

This case study shows a practical analytics workflow for evaluating a recruiting event:

- define success criteria (participation, experience, and follow-on intent)
- combine operational exports (registration / participation signals) with survey feedback
- produce **public-safe** deliverables that preserve decision value while removing identifying data

## What I did (first pass)

- Built **aggregated** survey summaries (Likert distributions and yes/no items).
- Built **aggregated** registration summaries (counts by industry, employment types, and target student years).
- Generated a short results snapshot to support “what should we change next time?”

## Outputs to read first

- `docs/Results_Snapshot.md`
- `README.md` (file map + how to reproduce)

## Data & privacy note

The original working exports can contain employer and registrant contact details. This public project package intentionally includes only aggregated tables in `data/`.
